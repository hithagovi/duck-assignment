# Duck Assignment

This project implements a Bayesian classifier to identify duck pixels in images.

## Files
- `bayes_duck_classifier_clean.py`: Main classifier script
- `collect_pixels_visible.py`: Script to collect pixel data
- `duck_pixels.npy`: Numpy array of duck pixel data
- `non_duck_pixels.npy`: Numpy array of non-duck pixel data

## Usage
Run the classifier script to analyze images for duck content.
